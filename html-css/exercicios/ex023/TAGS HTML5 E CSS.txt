Table - tabela

Table Header = th - cabeçalho

Hierarquia de tabelas Grandes:
THEAD - TR,TD,TH
TBODY - TR,TD,TH
TFOOT - TR,TD,TH

 text-align: left; - alinha o texto em: LEFT,CENTER,RIGHT
 vertical-align: bottom; - alinha o texto em: TOP,MIDDLE,BOTTOM.
